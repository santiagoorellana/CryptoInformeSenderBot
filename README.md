
<h1>Crypto Informe Sender Bot v1.1</h1>

<p>Esta es la version inicial que se utilizó como servicio mínimo viable, con la intención de atraer usuarios/clientes al canal de Telegram y empezar a medir la viabilidad del modelo de negocio.</p>

<p>Estos script se utilizan para realizar un análisis de una o más criptomonedas de forma automatizada y obtener el resultado como web local o verlos en un canal de Telegram. En un mismo informe web se pueden ver las medias SMA de las últimas 24 horas de la criptomoneda y acompañado de un texto explicativo y parámetros numéricos. Así se ve el gráfico del análisis de las últimas 24 horas:</p>
